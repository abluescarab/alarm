Introduction

Thanks for using Simple Alarm!

Simple Alarm is a simple, lightweight alarm that runs in your system tray and alerts you at a time of your choosing with a customized message.

Features:

  • Lightweight and easy-to-use
  • Simple controls
  • Plenty of options

Instructions

Simple Alarm is simple and easy to use. Follow these instructions to set an alarm!

Setting a Time

To set a time, just set the Hour, Minute, and Second to a time of your choosing. Note that it has to be a specific time, such as 15 seconds after 9PM. When the alarm goes off, a sound will play and a message will be displayed.

You can switch between a 24-hour and 12-hour clock in the Options menu.

Setting a Message

The message will appear in a popup window when the alarm goes off. If there is no message, a predetermined message will appear.

You can disable the message in the Options menu.

Status Bar

The status bar is the bar at the bottom of the window. It displays the current time and date and the time the alarm will go off.

You can disable showing either the time or date in the Options menu.

System Tray

The system tray is the section of your screen on the bottom right. When you minimize Simple Alarm, it will show an icon in the system tray. Right-click the tray icon and select "Show Window" to open the program again, or double-click the icon.

You can choose to show the icon all the time (rather than just when the program is minimized) in the Options menu.

Options Menu

The Options menu can be accessed by clicking "Options" at the top of the screen.

Always show tray icon

This will always show the tray icon, whether or not the program has been minimized.

Disable alert popup

This will disable the message popup. When the alarm goes off, only a sound will play (unless it has been disabled).

Disable sound alert

This will disable the sound alert. When the alarm goes off, only a message will show (unless it has been disabled).

24-hour clock

This will switch between 12-hour and 24-hour time. If checked, the program will display the time in 24-hour format.

Day/Month/Year format

This will switch between month/day/year format and day/month/year. If checked, the program will display the date in day/month/year format.

Hide date in statusbar

This will hide the date on the bottom of the window.

Hide time in statusbar

This will hide the time in the bottom of the window.
